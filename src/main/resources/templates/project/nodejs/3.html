프로젝트 개요

초등부 코딩 교육을 위한 블록형태의 웹기반 블록코딩 페이지 개발
드래그앤 드랍으로 블록을 Workspace에 가져다 놓으면, Arduino, Javascript 코드가 자동으로 생성되고 코딩한 블록들을 Run 버튼을 눌러 실행하면,
Javascript 코드가 실행되거나, 로봇이 움직임
블루투스 지원, 파일저장 등의 Utill 기능을 지원

주요 개발목표

"Blockly" 오픈소스를 활용하여 에코서버 형태의 웹페이지 개발

"Express" 모듈을 활용한 에코서버 구현

App, Router, Model, View 독립적 구조개발

블록을 쌓을때, 실시간 아두이노 코드로 변환(Arduino Interpreter)

블록명세 xml파일을 Customizing하여 카테고리 구현

"noBLE" 모듈을 활용한 웹 Bluetooth 구현

그 외에, 각종 Javascript 커스텀작업(이미지블록, GIF블록 등등..)
