프로젝트 개요

가입된 회원들에게 직원할인가를 추가로 적용하여, 저렴하게 휴대폰을 사용자들에게 제공하고, 선택한 요금제 및 옵션등의 견적을 한 눈에 볼 수 있는 웹사이트를 제작

해당 프로젝트는 유료 웹템플릿 소스코드 (HTML, CSS, SCSS, JS)를 구매하여, PHP 코드로 재구현하여 제작되었습니다.

주요 개발목표

MVC 패턴을 기반으로한 PHP 구조

'pdo' 모듈을 활용한 DB접근 제어

JQuery, Javascript를 활용한 할인품목, 옵션 선택시 실시간 가격 제공

SheetJS 라이브러리를 활용하여 요금제,단말기,할인가등 Excel로 관리

Excel <-> MysqlDB 포맷화

선택한 옵션 및 단말기의 견적을 DB에 저장하고 언제든 조회

'공유하기'를 통해 사용자의 견적서를 다른 사람에게 전송

유지 보수를 위한 관리자페이지 제작

새 요금제, 새 단말기 출시등 데이터 유지보수를 위한 스키마관계 정립
